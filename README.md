# Matrix sign

## About
This is a rewrite of an earlier coding project assignment written in Python that caused me nothing but problems as I was limited to what I was permitted to use at the time.

All aspects of the code had to be string orientated:

```
.**...***.
.*.*..*...
.**...***.
.**...*...
.*.*..*...
.*.*..***.
```

While I completed the entire project originally within three days using what I knew from my experience with Python:
- Dictionaries
- Lists
- Match/Case statements

I had not completed the project per the requirements, so I had to rewrite the entire codebase again, and even then it was not reflective of the requirements.

In the end, I rewrote the project about four times before it resembled the requirements, by then I was using git for tracking commits, and pushing them to Github.
